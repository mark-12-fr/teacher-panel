How It Works — Technical & Functional Guide
Teacher Management Portal  |  PHINMA University of Iloilo  |  June 2026
1. Overview — Ano ang AI Assistant?

Ang AcadTrack AI Assistant kay usa ka built-in nga intelligent chat tool sa Teacher Management Portal. Naga-analyze siya sang real-time student data (grades, attendance, missing requirements) kag naga-generate sang professional, emoji-formatted reports kag recommendations para sa teacher — lahat libre, wala bayad.

💡 Simple explanation: Ang teacher naga-ask sang question → ang AI naga-basang sang tanan nga student data → naga-generate sang professional na sagot with emojis, sections, kag recommendations — automatic, pila ka segundo lang.

  • Naga-call sang Groq/Gemini API
  • Naga-return sang formatted reply

🧠 ai-assistant.js (Frontend)

  • Naga-detect sang query intent
  • Naga-format sang AI reply (HTML + emojis)
  • Naga-manage sang chat persistence

🤖 AI Models

  • Fallback: Google Gemini (auto-switch)
  • Automatic fallback if Groq hits rate limit
  • Both are FREE tier
3. Algorithm — Pano Naga-Work ang AI Step by Step?
1
Teacher mag-type sang question sa chat Example: "Show me failing students" or "Sin-o ang absent subong?"
2
isEvaluationIntent() — nagatesting sang query Naga-check kung ang question kay about sa students, grades, attendance, etc. Kung yes → proceed to AI. Kung simple lang (general questions) → may basic response.
3
Fetch student data from Supabase Ang page naga-collect sang lahat nga data: students, sections, class records (grades/scores), attendance logs.
4
buildAIContext() — naga-build sang structured context Naga-compute sang final grade per student (WW 30% + PT 50% + Exam 20%), naga-list sang missing items, absent/late counts, kag naga-add sang specialized context base sa query type (at-risk, section comparison, etc.).
5
callAIEvaluate() — naga-send sa Flask backend POST request sa /api/ai-evaluate with { question, context }. Context limited to 4,500 characters para ma-fit sa token limit.
6
Flask naga-try Groq → fallback to Gemini Groq (fast) ang una. Kung mag-hit sang rate limit (6,000 tokens/min) → auto-switch sa Gemini (1M tokens/min). Teacher wala nakahibalo — seamless.
7
AI model naga-generate sang reply Naga-follow sang system prompt: bold emoji title, ## section headers, emoji bullets (✅❌⚠️📌📅), bold grades/names, kag 💡 Recommendation section.
8
formatAIText() — naga-convert sa HTML **bold** → <strong>, ## headers → styled div, bullet lists → <ul> with proper spacing.
9
Reply naga-appear sa chat widget Beautifully formatted, readable ang reply. Chat history naga-persist sa localStorage (same day lang).
10
Teacher pwede mag-export 3 export buttons sa chat header: 🔴 PDF, 🟢 Excel (.xlsx), 🔵 Word (.doc) — i-download ang buong conversation.
// Grade computation algorithm (ai-assistant.js) const ww = Math.round(Math.min(totalWrittenWork, 100)); // 30% weight const pt = Math.round(Math.min(totalPerfTask, 100)); // 50% weight const qe = Math.round(Math.min((examScore/50)*100, 100)); // 20% weight const grade = Math.round(ww * 0.30 + pt * 0.50 + qe * 0.20); // Passing: 75%+

6. Export Features — PDF, Excel, Word

After mag-ask sang question, ang teacher pwede mag-download sang AI conversation as:

🔴 PDF Export

  • Uses jsPDF library (loaded on-demand)
  • Formatted with title, page, timestamp
  • All messages with role labels
  • Multi-page support

🟢 Excel Export (.xlsx)

  • Uses SheetJS library (loaded on-demand)
  • Tabular format: Role, Message, Page, Date
  • Column widths auto-sized
  • Editable in Excel/Google Sheets

🔵 Word Document (.doc)

  • HTML-to-Word conversion (no library needed)
  • Color-coded bubbles (blue=teacher, green=AI)
  • Opens in MS Word, LibreOffice
  • Professional formatting

📍 Location of Buttons

  • AI chat header (top right area)
  • Always visible when chat is open
  • Toast notification on success/error
  • Libraries load only when needed
